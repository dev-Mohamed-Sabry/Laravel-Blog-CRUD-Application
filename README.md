# Laravel Blog (CRUD Application)

A simple blog application built with **Laravel** where users can create, read, update, and delete posts.  
This project was developed as a learning exercise to practice Laravel fundamentals, database relationships, validation, and MVC architecture.

---

## Features

- **CRUD operations**: Create, Read, Update, Delete posts
- **User association**: Each post is linked to a user
- **Form validation**: Ensures proper input data
- **Database migrations**: Easy setup for tables
- **Eloquent ORM**: Clean and efficient database interactions
- **Blade templates**: Dynamic views with reusable components
- **Bootstrap UI**: Simple and responsive user interface

---

## Technologies Used

- **PHP**  
- **Laravel**  
- **MySQL**  
- **Bootstrap 5**
